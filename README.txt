= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't even need to leave your web browser. To do an automatic install of Advanced Flat Rate Shipping Method, log in to your WordPress admin panel, navigate to the Plugins menu and click Add New.

In the search field type "Advanced Flat Rate Shipping Method WooCommerce" and click Search Plugins. Once you have found our plugin you can install it by simply clicking Install Now. After clicking that link you will be asked if you are sure you want to install the plugin. Click yes and WordPress will automatically complete the installation.

= Manual installation =

The manual installation method involves downloading our plugin and uploading it to your web server via your favorite FTP application.

<ol>
<li>Download the plugin file to your computer and unzip it</li>
<li>Using an FTP program, or your hosting control panel, upload the unzipped plugin folder to your WordPress installation's wp-content/plugins/ directory.</li>
<li>Activate the plugin from the Plugins menu within the WordPress admin.</li>
</ol>

== Frequently Asked Questions ==

= How do I set a product-based shipping method, for instance, a $10 fee if the cart contains a specific product?

To apply a $10 shipping fee for a specific product, such as "Happy Ninja," set up a rule as follows: Choose the condition where a cart contains the product "Happy Ninja" and set the shipping cost to $10.

= How can I set shipping fees based on both product category and country, such as a $10 charge for poster category products shipped to the UK? =

To set shipping charges for products in the "poster" category shipped to the UK, you'll need to create a rule where the cart contains a product from the "poster" category, then specify the country as the UK, and set the shipping cost at $10.

= What options do I have for displaying shipping methods on the checkout page? =

Shipping methods can be displayed as radio buttons or a dropdown. Adjust this setting in the Master Settings according to your preference.

= What should I do when multiple shipping methods apply to a cart? =

In the Pro version, the Master Settings provide several options:
 <ul>
 <li>Allow customer to choose: Lets the customer select from available shipping methods.</li>
 <li>Apply Highest: Only the most expensive shipping method is displayed.</li>
 <li>Apply smallest: Only the least expensive shipping option is shown.</li>
 <li>Force all: Charges the customer with the sum of all applicable shipping methods.</li>
 </ul.

= If the cart contains products from multiple categories with different shipping methods, how can the total shipping charge be calculated? =

For the Pro version, you can sum all applicable shipping charges by selecting "Force all shipping methods" in the Master Settings.

= How do I charge a shipping cost based on the total quantity of products in the cart? =

To charge a shipping fee based on the total quantity, use a formula like $10*[qty], where $10 is charged for each item in the cart.

= How can I set shipping methods for specific countries, states, and postcodes? =

In the Pro version, select the location-based shipping method rule, then specify the country, state, and postcode‚Äîfor example, USA, Alabama, Postcode 35801.

= How do I create a special shipping method for a specific user, such as "John"? = 

For creating a special $10 shipping method for "John," select "User equal to ‚ÄòJohn‚Äô" in the shipping rules.

= How can I apply a shipping charge based on the cart subtotal? For example, a $49 charge for orders under $99. =

Set a condition where if the cart subtotal is less than $99, the shipping charge will be $49.

= Can I display only the most expensive shipping method when multiple classes are involved? =

Yes, choose the calculation type as ‚ÄúPer order: charge shipping for the most expensive shipping class‚Äù to show only the most expensive option.

= Can this plugin accommodate weight-based flat rate shipping? =

Yes, the Flat Rate Shipping Plugin for WooCommerce supports weight-based shipping, allowing you to set flat rates based on the total weight of the cart.

= Are there options for per item and per order shipping rates? =

Absolutely, you can configure shipping rates on a per item or per order basis, giving you greater flexibility in how you apply shipping fees to your orders.

= How does the plugin enhance checkout speed and accuracy? =

By providing tailored shipping options based on your specific settings, the plugin simplifies the checkout process, leading to a smoother user experience and increased customer satisfaction.

= How does this plugin contribute to higher customer satisfaction? =

With strategic flat rate shipping options that match your business needs, the plugin ensures accurate shipping charges and efficient checkout, enhancing the overall shopping experience for your customers.
